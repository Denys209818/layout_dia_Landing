# Layout Dia Landing

## Project Overview

Layout Dia Landing is a landing page designed to highlight and promote the Air (formerly Dia) project. It features a sleek, responsive design and provides an engaging user experience.

## Technologies Used

- **Parcel** for project bundling
- **HTML** for page structure
- **CSS** (BEM methodology) for styling
- **JavaScript** for interactivity

## Additional Instructions for Setup

1. Clone the repository locally:
   ```bash
   git clone https://github.com/Denys209818/layout_dia_Landing.git
   ```
2. Navigate to the project directory:
   ```bash
   cd layout_dia_Landing
   ```
3. Install dependencies using npm:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
5. Open the following URL in your browser: [http://localhost:3000](http://localhost:3000).

> **Note**: Ensure that Node.js is installed on your system for successful setup. The server runs on port 3000.
